## Installation

```bash
npm i oxlint-fi -D
```

## Usage

```bash
npx oxlint-fi
```

### Static Build

If you want to build a static web app for Oxlint Formatter

```bash
npx oxlint-fi build
```

> This will generate a SPA under `.oxlint-formatter-inspector` folder. You can deploy it to any static file server.
